# UKL_Lat_2025
Jadi saya mengerjakan project Backend dari sekolah yaitu latihan UKL, jadi awal-awal saya menginisiasi projectnya terlebih dahulu yang saya tulis di terminal "npm init -y", lalu saya menginstall library terlebih dahulu untuk project saya seperti express, nodemon, prisma, md5, dontenv, jsonwebtoken, dan @prisma/client. 
Lalu setelah itu mengubah susunan dari package.json yang pertama saya rubah adalah tipe dari project saya yaitu tipe module saya menambahkan script dev yang menjalankan nodemon.
Lalu saya menginisiasi prisma, setelah itu saya mengisi schema prisma, di schema prisma saya membuat model database untuk menyimpan data yang ada di project ini.
Lalu setelah itu saya me-migrate prisma ke database, sebelum itu saya mengkoneksikan database lewat file ".env"
